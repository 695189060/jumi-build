## 🍊 桔米是什么

**桔米不是聊天机器人，是真正的 AI 数字员工。**

桔米能自己打开微信、浏览器、ERP 系统，像真人一样点击、输入、切换窗口、拖拽文件，自动完成您交代的工作任务。您只需要用自然语言下达指令，剩下的交给桔米。

> 💡 **一句话总结**：给桔米一个任务，它会像您的同事一样，打开电脑、操作软件、完成工作、汇报结果。

## 🏆 架构亮点

### 🧠 多模型智能路由 + 容灾
编码任务选代码模型、推理分析选推理模型、日常对话选性价比模型。**DeepSeek / 通义千问 / OpenAI 等多家异构兜底**，故障自动熔断切换，一家挂了不影响业务。

### 🎯 AI 意图识别
自动判别客户是闲聊、咨询还是有文档/报价需求，AI 先分类再决策。**规则 + AI 双重判定**，识别不准时自动回退，回复更贴合场景。

### 💬 边想边发流式回复
首句优先生成并立即发送，后续内容边想边补，配合分段发送。**回复节奏更像真人**，客户不用干等。

### 🗂️ 客户档案 + 联系人记忆
跨会话记住每位客户的姓名、需求、偏好和历史交互，自动沉淀客户档案。换个时间再聊，桔米依然记得上次聊到哪。

### 🔐 数据 100% 本地运行
所有客户资料、对话记录、工作文件**全部存储在您自己的电脑上**。支持本地大模型推理（4B/9B/35B），配合本地知识库检索，敏感任务可完全不出网。

### 🤖 真人级操作拟态
贝塞尔曲线鼠标轨迹、随机打字节奏、自然操作间隔。内置**账号防封机制和风控引擎**，三档风险模式可选，最大限度保护账号安全。

### 💡 越用越聪明，越用越省钱
独创本地知识碎片化缓存技术，基于本地 ONNX 向量化检索。桔米从每次对话中自动学习、沉淀企业知识资产。**相似问题命中即 0 token 秒回**，大幅降低 AI 调用费用。

### ⚡ 秒级响应，实时消息桥接
独创混合架构：消息接收走协议层（实时推送，<1 秒响应），消息发送走 RPA（模拟真人，防封号）。两种数字人交互模式自由切换：悬浮数字人 / DeepSeek 风格对话面板。

### 🖼️ 多模态理解
直接粘贴截图或上传文件，数字人即可识别分析。支持图片内容理解、文档摘要、数据提取。**对话即学习**，每次交互都在积累本地知识。

### 1. 下载安装
访问 [官网](https://www.jumidongli.cn) 下载 Windows 安装包（约 250MB），双击运行，按向导完成安装。

### 2. 首次启动
启动桔米后，系统会自动检测本机显卡并加载 AI 组件，请耐心等待。

> 💡 **提示**：如遇 Windows SmartScreen 提示，请点击「更多信息 → 仍要运行」

### 3. 设定工作区
设定桔米的工作区域，授权需要操控的应用程序（微信、浏览器等）。

### 4. 下达指令
用自然语言告诉桔米要做什么，就像跟同事交代工作一样。"帮我跟进这些客户"、"每天发一条朋友圈"。

### 5. 桔米自动干活
桔米在后台 7×24 小时工作。重要事项即时通知您，每天自动生成工作报表。

## ❓ 常见问题

<details>
<summary><strong>桔米会把我的客户数据上传到云端吗？</strong></summary>
不会。所有客户资料、对话记录、工作文件全部存储在您自己的电脑上。敏感任务可使用本地大模型推理，完全不出网。
</details>

<details>
<summary><strong>自动操作微信、抖音等平台，会不会导致账号被封？</strong></summary>
桔米内置了贝塞尔曲线鼠标轨迹、随机打字节奏、自然操作间隔等拟人化技术，配合三档风控模式（保守/正常/激进）和账号养护机制，最大限度保护账号安全。但我们建议遵守各平台的使用规范。
</details>

<details>
<summary><strong>需要什么样的电脑配置？必须联网吗？</strong></summary>
最低需要 Windows 10 64位、8GB 内存。联网状态下可使用云端 AI 模型获得最佳效果；基础本地缓存和部分任务支持离线运行。
</details>

<details>
<summary><strong>桔米和普通 AI 聊天机器人有什么区别？</strong></summary>
聊天机器人只能对话，桔米能实际操作电脑——点击、输入、切换窗口、操作各种软件。它是真正的"数字员工"而非"对话工具"。
</details>

<details>
<summary><strong>支持对接我们公司的 ERP / 电商系统吗？</strong></summary>
桔米内置了用友 T+、金蝶云、旺店通等系统连接器，也支持通用 REST API 和 AI 自动生成驱动，可对接绝大多数企业系统。
</details>
