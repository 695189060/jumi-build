# Security Policy

## 数据安全承诺

桔米 AI 数字员工的核心安全原则：

- **数据本地化**：所有客户资料、对话记录、工作文件默认存储在用户本地电脑，不上传云端
- **本地推理**：敏感任务支持完全离线运行，使用本地大模型（4B/9B/35B）
- **传输加密**：云端通信全程 TLS 1.3 加密
- **本地加密**：敏感数据使用 Windows DPAPI 系统级加密存储

## 安全架构

| 层级 | 措施 |
|------|------|
| 数据存储 | Windows DPAPI 加密，换机即失效 |
| 扩展运行 | 沙箱隔离，禁止越权访问 |
| 操作审计 | 关键操作不可篡改日志 |
| 异常保护 | 熔断机制，检测异常自动停止 |
| 权限控制 | 分角色权限，超授权自动拦截 |
| 工厂安全 | 设备超限自动急停，一键人工接管 |
