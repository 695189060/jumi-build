# 安装指南

## 系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| 操作系统 | Windows 10 64位 (21H2+) | Windows 11 64位 |
| 内存 | 8 GB | 16 GB+ |
| 存储 | 5 GB 可用空间 | 10 GB+ SSD |
| 显卡 | 支持 DirectX 11 | NVIDIA GTX 1060+ (本地模型推理) |
| 网络 | 宽带连接 | 宽带连接 |

## 下载安装

1. 访问 [GitHub Releases](../../releases) 页面
2. 下载最新版 Windows 安装包（约 250MB）
3. 双击运行安装包，按向导完成安装

> 💡 **提示**：如遇 Windows SmartScreen 提示，请点击「更多信息 → 仍要运行」

## 首次启动

1. 启动桔米，系统自动检测显卡并加载 AI 组件（首次较慢，请耐心等待）
2. 注册或登录账号
3. 设定工作区域，授权需要操控的应用程序
4. 开始用自然语言下达指令

## 常见安装问题

### SmartScreen 阻止运行
点击「更多信息」→「仍要运行」即可。所有安装包经过数字签名，安全无毒。

### 显卡驱动过旧
如遇 AI 模型加载失败，请更新显卡驱动到最新版本。

### 杀毒软件误报
部分杀毒软件可能误报 RPA 组件。请将桔米安装目录添加到白名单。

## 卸载

通过 Windows「设置 → 应用 → 桔米 → 卸载」即可完全卸载。卸载前可选择是否保留本地数据。
